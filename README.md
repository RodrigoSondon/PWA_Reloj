# ⏱ PWA Reloj

Una **Progressive Web App (PWA)** que muestra la hora y fecha exacta del sistema en tiempo real, con un diseño tipo display LED digital sobre fondo negro.

## Cómo ejecutar localmente

```bash
# Clonar el repositorio
git clone https://github.com/RodrigoSondon/PWA_Reloj.git
cd PWA_Reloj

# Levantar un servidor local en el puerto 5500
npx serve . -p 5500
```
Abrir el navegador en: **http://localhost:5500**
---

## Instalar como PWA

Una vez abierto en el navegador (Chrome / Edge / Safari en iOS):

- **Desktop:** aparecerá un ícono de instalación en la barra de direcciones
- **Móvil:** selecciona *"Agregar a la pantalla de inicio"* desde el menú del navegador
